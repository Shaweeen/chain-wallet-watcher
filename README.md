# Chain Wallet Watcher

[English](#english) | [中文](#中文)

---

## English

A powerful multi-chain wallet monitoring tool that tracks fund movements across Ethereum, BSC, Polygon, Solana, and Bitcoin networks with real-time Telegram notifications.

### Features

- **Multi-Chain Support**: Monitor wallets on Ethereum, BSC, Polygon, Solana, and Bitcoin
- **Real-time Monitoring**: Track incoming and outgoing transactions in real-time
- **Telegram Notifications**: Instant alerts via Telegram Bot
- **Configurable**: Easy YAML/JSON configuration for multiple wallets
- **High Performance**: Built with TypeScript for type safety and reliability
- **Extensible**: Easy to add new chains and notification channels

### Quick Start

```bash
git clone https://github.com/Shaweeen/chain-wallet-watcher.git
cd chain-wallet-watcher
npm install
cp config/config.example.yaml config/config.yaml
npm run build
npm start
```

## 中文

一个强大的多链钱包监控工具，支持实时追踪 Ethereum、BSC、Polygon、Solana 和 Bitcoin 网络上的资金流动，并通过 Telegram 即时推送通知。

### 功能特点

- **多链支持**: 监控 Ethereum、BSC、Polygon、Solana 和 Bitcoin 钱包
- **实时监控**: 实时追踪转入和转出交易
- **Telegram 通知**: 通过 Telegram Bot 即时推送提醒
- **灵活配置**: 支持 YAML/JSON 配置多个钱包地址
- **高性能**: 使用 TypeScript 构建，类型安全且可靠
- **可扩展**: 易于添加新链和通知渠道

### 快速开始

```bash
git clone https://github.com/Shaweeen/chain-wallet-watcher.git
cd chain-wallet-watcher
npm install
cp config/config.example.yaml config/config.yaml
npm run build
npm start
```
